HTML class & id

모든 html elements는 class와 id attribute를 갖을 수 있다. 이때 두 속성은 선택자로 이용된다.

복수의 선택자 class

모든 html 요소에 하나 이상의 이름을 가지는 것을 클래스라 한다. 이때 class는 중복가능하다.

  1. html : <tagname class="name1 name2 ..."></tagname>
  2. style : .name{ property: value; }
  3. js : document.getElementsByClassName("className")[n]
  4. 배열의 자료형 [] 예)["토마토", 30, 0.12, "런던"] 길이가 4인 배열
  5. class는 복수의 자료(Document Object)를 처리할 때 사용한다.

유일한 선택자 id

  1. html : <tagname id="idName"></tagname>
    id="idName" (가능o, 권장o)
    id="id_name" (가능o, 권장x)
    id="name" (가능o, 권장x)
    id="01idName" (가능o, 권장x)
  2. 아이디를 만들때는 단어와 단어의 조합으로 시작은 소문자, 조합되는 단어는 대문자로 작성한다.
    권장하는 방식이 document에서 유일할 가능성이 높고 가독성이 높다.
  3. style : #idName { property: value; }
  4. js : document.getElementById("idName")
