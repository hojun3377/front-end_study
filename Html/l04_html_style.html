HTML style css(Cascading Style Sheets)

style css는 레이아웃, 색상, 텍스트, 크기, 이미지, 배경, RWD(Responsive Web Design) 등 HTML의 요소가 화면에 표 시되는 모든 방식을 제공한다.

css 구문(Syntax)

  • selector { property: "value"; (x) }
  • selector { property: value; (o) }
  • 선택자 { 요소: 값; 요소: 값; 요소: 값; ... }
  • 예) h1 { color: red; background-color: gray; }
  • h1 {
    color: red;
    background-color: gray;
    color: black;
    }
    -> h1은 블랙이된다. (나중에 선언한 것이 우선 적용)
  • tagname style="property: 'value'; property: 'value' ... ;"

style HTML에 삽입하는 3가지 방법

  1. inline css: <tagname> style="property: value;"></tagname>
  2. internal css: <head>selector { property: value; }</head>
  3. external css: <head><link rel="stylesheet" href="url"></head>
  4. 요소에 동일한 스타일이 다르게 적용되었을때 우선순위 !important > 인라인(inline) > 내부(internal) > 외부(external)
