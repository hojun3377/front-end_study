HTML SVG ( Scalable Vector Graphics )

xml 형식의 벡터( vector )기반 그래픽을 정의하는 요소

  • svg는 확장 가능한 web용으로 벡터 그래픽을 나타낸다.
  • svg는 xml형식으로 그래픽을 정의한다.
    ex)<tagName/>
  • svg는 모든 요소와 모든 속성에 애니메이션을 정의할 수 있다.
    ( 시간에 따라 스타일이 변하는 것을 애니메이션이라 한다. )
  • svg는 W3C html을 표준화하는 곳에서 권장하는 태그(요소)

SVG rect (사각형)

  • width : 너비
  • height : 높이
  • stroke : 테두리 색상
  • stroke-width : 테두리 굵기
  • fill : 도형의 background 색상
  • opacity : 도형의 투명도
  • x : 좌측 상단 꼭지점을 기준으로 사각형의 x 좌표
  • y : 좌측 상단 꼭지점을 기준으로 사각형의 y 좌표
  • rx : 사각형 모서리 굴곡의의 x축 반지름
  • ry : 사각형 모서리 굴곡의의 y축 반지름

SVG line (선)

  • x1 : 선의 시작 x축 좌표
  • y2 : 선의 시작 y축 좌표
  • x2 : 선의 끝 x축 좌표
  • y2 : 선의 끝 y축 좌표

SVG circle (원)

  • cx : 원의 중심 x축 좌표 (default:0)
  • cy : 원의 중심 y축 좌표 (default:0)
  • r : 반지름

SVG ellipse (타원)

  • cx : 타원의 중심 x축 좌표
  • cy : 타원의 중심 y축 좌표
  • rx : 수평 반경
  • ry : 수직 반경
