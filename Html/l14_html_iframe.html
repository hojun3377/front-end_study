HTML iframe ( inline frame )

iframe 은 다른 웹 페이지를 추가할 수 있다.

src : 출력될 url, width : 너비, height : 높이
( 모든 요소의 스타일 속성은 css와 class로 동일하게 적용가능하다. )

iframe 은 스타일( style )이 적용되고 border를 바꿀 수 있다.
보통 border를 없애는 편이다.( border: none )

iframe은 name attribute를 가진다.
name은 a 요소의 target으로 지정할 수 있다.
a요소의 target: _self(default) - 현재 창에서 이동, _blank - 새 창에서 이동, iframeName - iframe에서 이동
ex) name: "targetTest"

target : targetTest
target : _self
target : _blank
href : #hrefSelectorTest ( id 선택자만 권장 )
