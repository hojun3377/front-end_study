inline의 박스 모델링

padding, border, outline 만 있기 때문에 box로 볼 수 없고, 가로 세로가 컨텐츠 크기만큼만 설정된다.

strong inline box

border-radius: <percentage>

백분율 값을 이용해 원의 반지름 또는 타원의 짧은 반지름과 긴 반지름을 지정
음의 값은 유효하지 않음

border-radius: <length>

length를 이용해 원의 반지름 또는 타원의 짧은 바지름과 긴 반지름을 지정
음의 값은 유효하지 않음

box-shadow

box의 그림자를 생성한다.
value: x-offset | y-offset | blur | color, ...
마찬가지로 text-shadow는 요소의 text에 그림자를 생성한다.

box-shadow: -5px -5px 10px black, 5px 5px 10px blue;
