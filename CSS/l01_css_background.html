배경 속성 선택



background-color: value
 해당 요소의 style property로 배경의 색상을 지정
value
  • rgb([0-255], [0-255], [0-255])
  • #[000000-ffffff]
  • hsl([0-360], [0-100]%, [0-100]%)
<img src=value alt=value>
 display: inline-block인 독립적인 노드 객체
value
  • src value: path
     img의 위치에 대한 경로
  • alt value: alternative text value
     img가 정상적으로 로드되지 않을 경우 출력되는 text
background-image: url("path1"), url("path2") ...
 해당 요소의 style property로 배경에 이미지를 삽입
value
  • path: img의 위치에 대한 경로
  • url: path를 지정해 위치를 찾아주는 css 함수
  • background-repeat: value
    (default: repeat)
    background-image가 반복할지 정한다.
    • repeat: 배경 이미지를 반복한다.
    • no-repeat: 배경 이미지를 반복하지 않는다.
  • background-size : width height
    background 의 다운받은 이미지 전부의 크기를 설정한다.
    default: auto
    • contain : 이미지 비율을 유지하면서 노드의 크기만큼 자동으로 크기 조절한다.
    • cover : 이미지 비율을 유지하면서 노드의 요소를 최대한 가리기 위해 크기를 조절한다.
      (이미지가 잘린다, 부족한 축을 auto, 다른 축을 100%를 주는 것이다.)
  • background-clip: value
    value는 배경이 나타내는 영역이다.
    default: border-box
    • padding-box: padding을 포함한 영역
    • content-box: padding을 제외한 영역
    • border-box: border를 포함한 영역
  • background-origin
    value는 배경 이미지가 나타내는 영역이다.
    origin은 background-image가 적용된 경우에만 동작한다.
    default: padding-box
    • padding-box: padding을 포함한 영역
    • content-box: padding을 제외한 영역
    • border-box: border를 포함한 영역
  • background-position
  • background-attachment
