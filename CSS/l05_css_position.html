position

요소, 노드 개체의 위치를 정의한다.
static, relative, absolute, fixed, stiky

position: static

모든 요소 노드 개체는 위치를 static으로 정의한다.
position: static ( default )

  • 요소의 배치는 순차적이며 ( 위에서 부터 아래로 ) 좌에서부터 우로 배치된다.
  • 자식요소는 부모요소의 위치를 기준으로 배치된다.
  • top, bottom, left, rigth를 사용할 수 없다.

position: relative ( 상대적 위치 )

자식 박스를 상대적 위치로 왼쪽 위 100px 씩 이동

  • 상대적 위치 relative는 정적위치 static을 기준으로 노드를 이동시킨다.
  • 상대적 위치로 이동한 노드 개체는 다른 노드에 영향을 주지 않는다.

position: absolute ( 절대적 위치 )

부모가 static

자식 박스를 상대적 위치로 왼쪽 위 100px 씩 이동

부모가 relative

자식 박스를 상대적 위치로 왼쪽 위 100px 씩 이동

  • 상위 노드 중에 relative, absolute, fixed인 개체가 있을 경우 그 개체를 기준으로 위치를 지정한다.
  • 상위 노드 중에 relative, absolute, fixed인 개체가 없을 경우 body를 기준으로 위치가 지정된다.
  • 부유 객체로 다른요소를 밀거나 당기지 않고 덮어쓰는 형태로 존재한다.

position: fixed

부모가 relative이고 overflow: auto인 노드의 fixed 된 제목
