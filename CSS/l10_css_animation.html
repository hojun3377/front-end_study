CSS animation으로 만드는 애니매이션

animation은 javascript의 setinterval로 만드는 동작을 간단하게 작성할 수 있다.

  • animation: name duration ... : 모든 프로퍼티를 작성 ( shorthand syntax, 권장x )
  • animation-name: name : @keyframes 이름을 지정
    이벤트로 애니매이션을 등록할 때 등록할 요소에 animation-name을 추가한다.
    만약 동작하지 않으면 이름이 정의된 클래스를 추가하면 된다.
  • @keyframes animation-name : 애니매이션이 동작하는 단계를 나눠서 설정
  • animation-delay: time : 요소가 시작될 시점 ( delay < 0인 경우 : 애니매이션이 동작된 미래를 나타낸다. )
  • animation-duration: time : 한 싸이클의 애니매이션이 소유되는 시간 ( s | ms, default 0 )
  • animation-timing-function: timing-function : 애니매이션의 속도를 나타내는 함수 ( default ease )
  • animation-iteration-count: count : 애니매이션의 재생 횟수 ( infinite: 무한히 반복, default 1 )
  • animation-play-state: state : 애니매이션의 재생 상태 ( running | paused, default: running )
  • animation-direction: direction : 애니매이션의 방향
    direction value
    • normal(default) : form->to
    • reverse : to->from
    • alternate : 홀수번 cycle - normal & 짝수번 cycle - reverse
    • alternate-reverse : 홀수번 cycle - reverse & 짝수번 cycle - normal
  • animation-fill-mode: mode : 애니매이션 실행 전과 후에 대상에 스타일을 적용하는 방법을 지정
    mode value
    • none(default) : 원래 스타일을 적용하고 animation-delay기간 동안 이 값을 유지
      애니매이션이 종료되면 원래 상태로 되돌리고 종료
    • forwards : 마지막 keyframe에 의해 설정된 계산된 값을 유지
    • backwards : 첫 keyframe에 정의된 값을 적용하고 animation-delay기간 동안 이 값을 유지
    • both : forwords와 backwords 모두 적용
