둘 다 값이 정해지지 않은 형태로, undefined는 초기화 되지않은(선언되지 않은) 상태고 null은 'type'이 정해지지 않은 빈상태이다.

기본데이터 타입은 아니지만, 기본데이터 타입처럼 사용할 수 있는 배열과 json

Array(배열) 타입

  • let arr=[]; : new 없이 생성 가능한 배열타입
  • let arr=new Array(a,b,c,d,...);

배열은 순서를 가지는 데이터타입으로 모든 데이터 타입을 참조할 수 있다. ( 같은 데이터 타입을 참조하는 것을 권장한다. )

JSON 타입

  • let json={key1:value1, key2:value2, ...}; : new 없이 생성 가능한 JSON타입
  • let json=new Object();

json은 key와 value를 가지는 데이터 타입으로 new Object로 생성 가능하다. 이때, Object는 모든 클래스의 부모인 Object와 별개의 클래스이다.

JSON은 JavaScript Object Notation의 줄임말로, 경량 데이터 교환 형식이다.
언어에 독립적으로 문자열 형식이 중요하다.
