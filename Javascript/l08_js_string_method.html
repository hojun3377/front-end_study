자바스크립트 문자열 String의 메소드

문자열은 유저에게 전달하는 데이터로 웹상에서 가장 많이 사용된다. 때문에 문자열을 다루는 함수가 많이 존재한다.

method description ex
toUpperCase()
toLowerCase() 		문자열의 대문자 or 소문자로 변환 "Ab Cd".toUpperCase() :
"Ab Cd".toLowerCase() :
trim() 문자열의 앞뒤 공백을 제거한다. " abcd ".length :
" abcd ".trim().length :
charAt(index)
charCodeAt(ASCIIcode) 		문자열 index의 문자(char)의 문자 반환
문자열 index의 문자(char)의 ASCIIcode 반환 		"ABCD".charAt(0) :
"ABCD".charCodeAt(0) :
slice(start, end=length) 일부 문자를 추출할 때 사용한다.
  • 양수 index : 0~string.length-1
  • 음수 index : -string.length~-1
  • start ~ end-1 문자 추출
  • 매개변수가 1개 : end는 string.length
  • start 위치는 end 위치보다 좌측에 위치
 "스파게티 피자 샌드위치".slice(8,12) :
"스파게티 피자 샌드위치".slice(8,-1) :
"스파게티 피자 샌드위치".slice(-4,-1) :
"스파게티 피자 샌드위치".slice(8) :
"스파게티 피자 샌드위치".slice(-4) :
"스파게티 피자 샌드위치".slice(12,8) :
substring(start, end=length) slice와 동일하지만 양수 index만 사용 가능하다.
하지만 slice와 다르게 start 위치가 end 위치보다 우측에 위치해도 start와 end 사이의 문자열을 출력한다. 		"스파게티 피자 샌드위치".substring(8,12) :
"스파게티 피자 샌드위치".substring(8,-1) :
"스파게티 피자 샌드위치".substring(-4,-1) :
"스파게티 피자 샌드위치".substring(8) :
"스파게티 피자 샌드위치".substring(-4) :
"스파게티 피자 샌드위치".substring(12,8) :
substr(start, len) 일부 문자를 추출한다. 이때 시작점부터 길이만큼 반환한다. len는 양수 "스파게티 피자 샌드위치".substr(8,4) :
"스파게티 피자 샌드위치".substr(-4,4) :
"스파게티 피자 샌드위치".substr(8,-1) :
"스파게티 피자 샌드위치".substr(-4,-1) :
"스파게티 피자 샌드위치".substr(8) :
"스파게티 피자 샌드위치".substr(-4) :
startsWith(str, position=0) 시작점부터(좌->우) 특정 문자열(str)로 시작하는지 여부 ( boolean ) "스파게티 피자 샌드위치".startsWith("스파") :
"스파게티 피자 샌드위치".startsWith("피자",5) :
endsWith(str, position=length) 끝에서부터(우->좌) 특정 문자열(str)로 시작하는지 여부 ( boolean ) "스파게티 피자 샌드위치".endsWith("샌드위치") :
"스파게티 피자 샌드위치".endsWith("샌드") :
"스파게티 피자 샌드위치".endsWith("피자",7) :
includes(str, position=0) 특정 위치(position) 부터 특정 문자열(str)이 있는지 여부를 검색 ( boolean ) foodList.includes("피자") :
foodList.includes("피자",6) :
indexOf(str, position=0) 특정 위치(position) 부터 특정 문자열(str) 이 있는지 검색 후 시작 지점의 index를 반환하고 탐색을 멈춤 ( 시작지점부터 찾음 )
string이 없다면 -1 반환 		"Brave new world".indexOf("new") :
"Brave new world".indexOf("new",7) :
"Brave new world".indexOf("hello") :
lastIndexOf(str, position=length) 특정 위치(position) 부터 특정 문자열(str) 이 있는지 검색 후 시작 지점의 index를 반환하고 탐색을 멈춤 ( 끝지점부터 찾음 )
string이 없다면 -1 반환 		"Brave new world".lastIndexOf("new") :
"Brave new world".lastIndexOf("new",5) :
split(str, arrayLen) 특정 문자열(str)을 기준으로 나누어 배열로 만든다.
arrayLen : 배열에 넣을 나눠진 문자열의 개수
만약 str이 기존 문자열에 포함된 문자열이 아니라면 기존 문자열을 원소로 가지는 문자열을 생성한다. 		"스파게티 피자 샌드위치".split(" ") :
"스파게티 피자 샌드위치".split(" ",2) :
"스파게티 피자 샌드위치".split(",") :
replace(str1, str2) 특정 문자열이나 정규식 패턴으로 검색 후 다른 문자열로 변환한 값을 반환
**새로운 문자열을 반환 ( 즉, 기존 문자열을 변환하지 않는다. )
**기존 문자열에서 가장 앞에있는 하나의 str1만 str2로 변환한다. 		"스파게티 피자 샌드위치".replace("샌드위치","햄버거") :
"스파게티 피자 샌드위치".replace(" ",",") :
replaceAll(str1, str2) 특정 문자열이나 정규식 패턴으로 검색 한 모든 문자열을 다른 문자열로 변환한 값을 반환
**새로운 문자열을 반환 ( 즉, 기존 문자열을 변환하지 않는다. )
**기존 문자열에서 모든 str1을 str2로 변환한다. 		"스파게티 피자 샌드위치".replaceAll(" ",",") :
