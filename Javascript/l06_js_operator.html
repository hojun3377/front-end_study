할당 연산자와 증가 및 감소 연산자

operator(연산) name(이름) description(설명) ex(예시)
a+=b 추가할당 a=a+b : 자기 자신 a에 b를 더한 값을 대입한다. x+=10 ( x=x+10 )
a-=b 빼기할당 a=a-b : 자기 자신 a에 b를 뺀 값을 대입한다. x-=10 ( x=x-10 )
a*=b 곱셈할당 a=a*b : 자기 자신 a에 b를 곱한 값을 대입한다. x*=10 ( x=x*10 )
a/=b 분할할당 a=a/b : 자기 자신 a에 b를 나눈 값을 대입한다. x/=10 ( x=x/10 )
a%=b 나머지할당 a=a%b : 자기 자신 a에 b를 나누고 남은 값을 대입한다. x%=10 ( x=x%10 )
++a; or --a; 증감연산자(전위) 전위 증감 연산자는 연산을 먼저 하고 문장을 실행한다. ( a=a+1; or a=a-1; )
ex) a(++i); : i를 1증가시키고 a()를 실행 		++x ( x=x+1 )
a++; or a--; 증감연산자(후위) 후위 증감 연산자는 문장을 실행하고 연산을 한다. ( a=a+1; or a=a-1; )
ex) a(i++); : a()를 실행시키고 i를 1증가 		x++ ( x=x+1 )
