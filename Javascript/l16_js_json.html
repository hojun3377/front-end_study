JSON ( JavaScript Object Notation )

  • 자바스크립트의 객체 표현법
  • JSON은 문자열로 표현될 수 있다.
  • 경량 데이터 교환 방식으로 문자열로만 이루어진다. ( 프로그래밍 언어에 독립적이다. )
  • 데이터를 주고 받기위해 사용된다.

JSON 작성법

  • JSON : "{ ... }"
  • const jsonData = "{ ... }";
  • const jsonData = "{ \"key1\":\"value1\", \"key2\":\"value2\", ... , \"keyN\":\"valueN\" }";
    이때 keyN은 중복될 수 없다.
    문자열로 이루어진 데이터형식이기 때문에 기본 데이터 타입(string,data,boolean,null, ... )과 배열(array)를 제외하고 모두 문자열로 작성한다.
  • json 형식의 문자열을 자바스크립트에서 JSON.parse()를 이용하여 Object 타입으로 변경한다.
  • json의 key는 property이기 때문에 .으로 접근이 가능하다.
  • 프로그래밍에서 key가 유동으로 바뀔때는 ["key"] 방식으로 접근가능하다.

자바스크립트가 JSON을 대하는 태도

  • 자바스크립트에서 json은 Object()의 property 명세서
  • new Object()로 json을 생성할 수 있다.
  • 자바스크립트에서 json을 기본 데이터 타입처럼 리터럴한 방식을 제공한다.
  • { key:value } : 리터럴 방식 ( 자바스크립트 문법에서 사용되는 json )
  • "{ \"key\":\"value\" }" : 문자열로 표현된 json ( 서버에서 주고받는 데이터 )
  • 자바스크립트는 함수지향언어로 함수를 객체로 선언할 수 있다.
  • function Person(name,age) { this.name=name; this.age=age; }
    => new Person("최호준",25) = Person : {name:"최호준",age=26}
  • 자바스크립트 객체의 property 설명서가 문자열로 데이터가 전송되는 것을 json이라 한다.

자바스크립트에서 json을 사용하는 법

  1. 함수로 JSON 생성
    function ObjectName(value1,value2, ... ,valueN) {
      this.key1=value1;
      this.key2=value2;
      ...
      this.keyN=valueN; }
  2. 함수로 JSON 생성
    function ObjectName(key1,key2, ... ,keyN) {
      this.key1;
      this.key2;
      ...
      this.keyN;
       }
  3. ( 리터럴방식 )
    const objectName = {
      key1:value1,
      key2:value2,
      ...
      keyN:valueN
    }
  4. (권장 x)
    const objectName = new Object();
    objectName.key1=value1;
    objectName.key2=value2;
      ...
    objectName.keyN=valueN;
  • json은 toString()을 매번 재정의해야하기 때문에 출력되지 않는다.
  • JSON.stringify()로 출력된 문자열을 서버 통신에 json으로 사용하면 된다.
