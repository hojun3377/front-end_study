자바스크립트 조건문

블럭(block) 안의 실행을 조건에 따라 동적으로 선택되게하는 조건문 if, switch, 삼항연산자

if 조건문

if( condition ){ block } : condition이 true면 block을 실행한다.

if( condition ){ block1 } else{ block2 } : condition이 false면 block2를 실행한다.

if( condition1 ){ block1 } else if( condition2 ) { block2 } else { block3 } :
condition1 검사 후 만족하면 block1 실행, 그렇지 않으면 condition2 검사 후 만족하면 block2 실행, 그렇지 않으면 block3 실행

  • if(value < 50) { ... }
  • if(value < 50 && value > 20) { ... }
  • if(value > 50 || value < 20) { ... }
  • if(value > 10 && value < 20 || value > 30 && value < 40) { ... }

switch 조건문

      작성법
          switch(expression) {
            case value1:
              ...
              [break]
            case value2:
              ...
              [break]
            case value3:
              ...
              [break]

            ...

            default:
              ...
          }

switch case는 엄격한 비교( === )을 사용하고, 위에서부터 일치하는 항목을 찾으면 switch는 종료된다.
( 엄격한 동등 비교 외의 조건을 사용할 수 없다. )

switch문을 사용하는 이유 (if 대신 switch를 사용하는 이유)

  1. if에서 else if 문과 동일하게 작동하지만, 조건문의 수행속도가 굉장히 빠르다.
  2. 조건을 파악하기 굉장히 좋다.

삼항연산자

작성법

  1. (condition) ? result1 : result2 ( == if( condition ){ true block } else{ false block } )
    ㄴcondition이 true이면 result1, false면 result2를 반환
  2. let valriable = ( condition ) ? trueObj : falseObj;
  3. (condition) ? result1 : ( (condition2) ? result2 : ( (condition3) ? result3 : result4 ) )
    ( == if(condition){ result1 } else if(condition2){ result2 } else if(condition3){ result3 } else{ result4 } )

조건 파악이 편하고 코드가 간결해 프로그래밍 속도를 올려준다.
