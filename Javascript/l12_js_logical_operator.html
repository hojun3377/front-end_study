자바스크립트의 논리연산자

&& (and), || (or), ! (not)

논리연산의 결과는 boolean 값으로만 나오지 않는다.

|| (or)

합집합을 의미하며 모두가 거짓이라도 하나만 진실이면 진실을 의미한다. 다른말로 모두 거짓이어야 거짓이다.

&& (and)

교집합을 의미하며 모두가 진실이라도 하나만 거짓이면 거짓을 의미한다. 다른말로 모두 진실이어야 진실이다.

논리연산의 순서(우선순위)

&&, || 는 우선순위가 존재한다. &&는 교집합으로 곱하기처럼 우선순위를 가진다. ||는 합집합으로 덧셈처럼 우선순위를 가진다.

논리 연산 문제

&&연산시 선택 순서 : NaN -> null -> undefined

! (not)

여집합을 의미하며 모두가 진실이라도 하나만 거짓이면 거짓을 의미한다. 다른말로 모두 진실이어야 진실이다.
