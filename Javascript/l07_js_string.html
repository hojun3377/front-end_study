문자들의 집합 문자열 ( String )

String 객체는 문자들의 집합인 문자열을 조작하는 함수를 제공한다.

문자열 생성 방법

  • let x = "abc"; ( string은 기본데이터 타입이다. )
  • let x = 'abc'; ( javascript는 char 타입이 없다. )
  • let x = `abc`; ( 템플릿 리터럴 ( Template literals ) 적용 문자 )

문자(char) 접근

  • "string"[index]
    ex) "string"[2] :
    **string을 배열처럼 index로 접근 할 때, read만 가능하다.
    ex) "string"[2] = "a"; (x)
  • "string".charAt(2) :

기본 데이터 타입인 문자열과 인스턴트 객체인 문자열

  • init
    let stringVal = "안녕하세요"; ( 기본 데이터 타입인 문자열 )
    let objectVal = new String("안녕하세요"); ( new 연산자로 생성한 인스턴트 객체 )
  • type
    typeof stringVal :
    typeof objectVal :
  • compare
    stringVal == objectVal :
    stringVal===objectVal :
  • 기본데이터 타입인 문자열에는 함수가 존재하지 않는다. 하지만 사용할 수 있는 이유는 사용할 때 인스턴트 객체 생성과 비슷한 일이 발생하기 때문이다.

escape sequence ( 이스케이프 시퀀스 )

특수 문자나 백슬러시(\), 줄바꿈(\n), 따옴표(") 등의 것을 출력 할 수 있다.

Template literals ( 템플릿 리터럴, ${expression(표현식)} )

문자열을 ``(백틱)으로 선언하면 ${expression} (Template literals) 표현식을 사용한다는 선언이다.
이때, 자바스크립트 문법을 expression에 사용할 수 있다.
